A. Matrixs

A.1 Basic Operations

For a real matrix A∈Rm×n , the element in the i -th row and j -th column is denoted as (A)ij=Aij . The transpose of matrix A is written as AT , where (AT)ij=Aji . Clearly:

(A.1)(A+B)T=AT+BT,
(A.2)(AB)T=BTAT.

For a matrix A∈Rm×n , if m=n , it is called an n -order square matrix. Let In denote the n -order identity matrix. The inverse matrix A−1 satisfies AA−1=A−1A=I . It can be easily shown:

(A.3)(AT)−1=(A−1)T,
(A.4)(AB)−1=B−1A−1.

For an n -order square matrix A , its trace is the sum of the elements on its main diagonal, i.e., tr(A)=∑i=1nAii . The following properties hold:

(A.5)tr(AT)=tr(A),
(A.6)tr(A+B)=tr(A)+tr(B),
(A.7)tr(AB)=tr(BA),
(A.8)tr(ABC)=tr(BCA)=tr(CAB).

The determinant of an n -order square matrix A is defined as:

(A.9)det(A)=∑σ∈Snpar(σ)A1σ1A2σ2…Anσn,

where Sn is the set of all n-order permutations, and par(σ) takes the value -1 or +1 , depending on whether σ=(σ1,σ2,…,σn) is an odd permutation or an even permutation—that is, the number of inversions is odd or even, respectively.

For an n -order square matrix A , the determinant has the following properties:

(A.10)det(cA)=cndet(A),
(A.11)det(AT)=det(A),
(A.12)det(AB)=det(A)det(B),
(A.13)det(A−1)=det(A)−1,
(A.14)det(An)=det(A)n.

For a matrix A∈Rm×n , the Frobenius norm is defined as:

(A.15)∥A∥F=(tr(ATA))1/2=(∑i=1m∑j=1nAij2)1/2.

It is easy to see that the Frobenius norm of a matrix corresponds to the L2 -norm of the matrix when treated as a vector.

A.2 Derivative

The derivative of a vector a with respect to a vector x , and the derivative of x with respect to a , are both vectors. The i -th component is given by:

(A.16)(∂a∂x)i=∂ai∂x,
(A.17)(∂x∂a)i=∂x∂ai.

Similarly, for a matrix A , the derivative with respect to a vector x , and the derivative of x with respect to A , are both matrices. The element in the i -th row and j -th column is given by:

(A.18)(∂A∂x)ij=∂Aij∂x.
(A.18)(∂x∂A)ij=∂x∂Aij.

For a function f(x) , assuming the elements of the vector x are differentiable, the first-order derivative of f(x) with respect to x is a vector, with the i-th component defined as:

(A.20)(∇f(x))i=∂f(x)∂xi.

The second-order derivative of f(x) with respect to x is called the Hessian matrix, where the i -th row and j -th column element is defined as:

(A.21)(∇2f(x))ij=∂2f(x)∂xi∂xj.

The derivatives of vectors and matrices satisfy the product rule. Specifically:

(A.22)∂(xTa)∂x=∂aTx∂x=a,
(A.23)∂(AB)∂x=∂A∂xB+A∂B∂x.

Here a is a constant vector compared to x

From A−1A=I and Equation (A.23), the derivative of the inverse matrix can be expressed as:

(A.24)∂A−1∂x=−A−1∂A∂xA−1.

If the target of differentiation is the elements of matrix A , we have:

(A.25)∂tr(AB)∂Aij=Bji,
(A.26)∂tr(AB)∂A=BT.

We further have:

(A.27)∂tr(ATB)∂A=B,
(A.28)∂tr(A)∂A=I,
(A.29)∂tr(ABAT)∂A=A(B+BT).

From Equations (A.15) and (A.29), we obtain:

(A.30)∂∥A∥F2∂A=∂tr(AAT)∂A=2A.

The chain rule is an essential tool for computing derivatives of composite functions. Simply put, for functions f , g , and h , where f(x)=g(h(x)) , we have:

(A.31)∂f(x)∂x=∂g(h(x))∂h(x)⋅∂h(x)∂x.

For example, when computing the following, treating Ax−b as a whole simplifies the calculation:

(A.32)∂∂x(Ax−b)TW(Ax−b)=∂(Ax−b)∂x⋅2W(Ax−b)=2ATW(Ax−b)..

A.3 Singular Value Decomposition

Any real matrix A∈Rm×n can be decomposed as:

(A.33)A=UΣVT,

where:

  • U∈Rm×m is an orthogonal matrix (unitary matrix in the real case), satisfying UTU=I ,

  • V∈Rn×n is also an orthogonal matrix, satisfying VTV=I ,

  • Σ∈Rm×n is a diagonal matrix with elements (Σ)ii=σi and all other elements equal to 0 . The singular values σi are non-negative real numbers that satisfy: σ1≥σ2≥⋯≥0.
